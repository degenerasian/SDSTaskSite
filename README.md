# SDSTask Site
Website for creating and assigning tasks to users, recording work hours, and managing projects, made for the purposes of SD Solutions IT Outsourcing Inc.

## Functions
### Project Creation
Create projects and configure user access
![Project Creation Page](https://github.com/degenerasian/SDSTaskSite/blob/main/Documentation%20Images/create%20project.png)<br><br>

### Task View
Create and assign tasks to users and view tasks in column or list form
![Create Task](https://github.com/degenerasian/SDSTaskSite/blob/main/Documentation%20Images/create%20task.png)
![Project Task List](https://github.com/degenerasian/SDSTaskSite/blob/main/Documentation%20Images/project%20tasks.png)
![Project Task Columns](https://github.com/degenerasian/SDSTaskSite/blob/main/Documentation%20Images/task%20column.png)<br><br>

### Manage Tasks
Edit task details, assignees, labels, and leave comments
![Edit Task](https://github.com/degenerasian/SDSTaskSite/blob/main/Documentation%20Images/task%20overview.png)
![Task with Comment](https://github.com/degenerasian/SDSTaskSite/blob/main/Documentation%20Images/task%20with%20comment.png)<br><br>

### Gantt Chart
Create a Gantt chart of projects on demand
![Gantt Chart of Project](https://github.com/degenerasian/SDSTaskSite/blob/main/Documentation%20Images/gantt%20chart.png)
